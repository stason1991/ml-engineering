# Career Path PRO



Проект включает в себя пользовательский интерфейс, бэкенд на FastAPI, асинхронную обработку через RabbitMQ, ML-воркеры и комплексную систему мониторинга.

## Конфигурация

Перед запуском необходимо создать файл .env в корневой директории. Вы можете скопировать настройки из примера:
```bash
cp .env.example .env
```
Файл содержит настройки базы данных, параметры подключения к RabbitMQ, секретные ключи для JWT-авторизации и параметры ML-моделей.

## Быстрый старт

1. Запуск всей инфраструктуры:
   ```bash
   docker-compose up -d
   ```

2. Доступ к интерфейсам:
   - Пользовательский интерфейс (Django): http://localhost:8000
   - Документация API (Swagger): http://localhost:8080/api/docs/
   - Панель управления RabbitMQ: http://localhost:15672 (данные из .env, по умолчанию guest/guest)
   - Prometheus (Сырые метрики): http://localhost:9090
   - Grafana (Дашборды): http://localhost:3000 (admin / admin)

## Тестирование

### 1. Функциональное тестирование (Pytest)
Тесты запускаются локально и проверяют API, транзакции и интеграцию с RabbitMQ.

Запуск:
```bash
pip install -r tests/requirements.txt
pytest tests/
```

### 2. Нагрузочное тестирование (k6)
Сценарий k6 запускается в изолированном контейнере, 
имитирует регистрацию, пополнение баланса и отправку данных на предсказание под нагрузкой 50+ пользователей.

Запуск нагрузочного теста:
```bash
docker-compose run --rm k6 run /scripts/load_test.js
```

## Мониторинг и аналитика (Prometheus & Grafana)

Система автоматически собирает технические и бизнес-метрики.

### Доступ к Grafana
- Адрес: http://localhost:3000
- Логин: admin | Пароль: admin
- Настроен анонимный вход (режим Viewer) для быстрого доступа к графикам.

### Что отслеживается:
- Фронтенд и Бэкенд: доступность сервисов и скорость обработки запросов.
- RabbitMQ: состояние очередей и скорость обработки ML-задач.
- Бизнес-метрики: количество успешных предсказаний и объем транзакций.
- Система: нагрузка на CPU, RAM и дисковое пространство.

## Технологический стек

- Frontend: Django (Python/JS).
- Backend: FastAPI (Python 3.11+), SQLAlchemy 2.0.
- Database: PostgreSQL 15.
- Task Broker: RabbitMQ 3.12 (с плагинами Management и Prometheus).
- Monitoring: Prometheus, Grafana, Node Exporter.
- Load Testing: k6 (JavaScript).

## Структура проекта

- /frontend_prod — пользовательский интерфейс на Django.
- /app — исходный код основного API на FastAPI.
- /worker — ML-воркеры для асинхронной обработки задач.
- /monitoring — конфигурации Prometheus и предустановленные дашборды Grafana.
- /tests — функциональные тесты (Pytest) и сценарии нагрузки (k6).
- /web-proxy — конфигурация Nginx (обратный прокси).
